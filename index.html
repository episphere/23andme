This is a data science research application, not for clinical use.

Interoperation with consumer genomics services

This Web application explores the use of 23andMe's API as to assess individual genetic risk by reference Polygenic Risk Score models maintained by PGS Catalog [notebook]. This is achieved by moving the risk calculation code to the user's device, rather that having the user's sensitive genomic data leave the user's own device. As a result, both sensitive information and prediction both stay with the user: the code calculating the risk does the travelling while the data stays put.
